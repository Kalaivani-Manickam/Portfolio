Badaga Speech-to-Text Translator

A deep learning project focused on the Badaga language, which is low-resource. - Built a speech-to-text system with spectrogram feature extraction. - Used RNN/LSTM models for sequence prediction. - Translates spoken Badaga into English text. This work aims to preserve and promote the Badaga language digitally.

Skin Cancer Data Visualization & Detection

This project combines data visualization with deep learning for healthcare. - Analyzed skin cancer datasets with graphs and visual dashboards. - Built a CNN model to classify dermoscopic images into benign/malignant classes. - Integrated with Gradio to allow users to upload images and get real-time predictions. This shows the practical use of AI in medical diagnostics.
